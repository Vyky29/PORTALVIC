Roster review — tablas editables

Para participantes y notas de pool, usa el grid MADRE (más fácil de ver y editar): roster_term_master_review.html. Aquí: turnos de staff (CSV) y tabla plana de participantes.

Después de editar:
  1. Descarga el CSV de la pestaña que cambiaste.
  2. Reemplaza el archivo en database/roster_review/ (mismo nombre).
  3. Participantes/notas: python database/roster_review/apply_participants_by_day_csv.py
  4. Staff: guarda staff-shifts.csv y pide en Cursor que lo aplique al timetable.
